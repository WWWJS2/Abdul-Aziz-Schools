صفحة المدرسة

  • اسم المدرسة والبرنامج.

  • وسائل التواصل ومعلومات الصفحة.

الخطوة ٤

تفاصيل ولي الأمر

في هذه المرحلة تُستكمل بيانات ولي الأمر الأساسية داخل النموذج بحسب ما تسمح به المنصة بعد خطوة التحقق.

ما يتم استكماله

  • البيانات الشخصية الأساسية
  • معلومات التواصل
  • أي بيانات تنظيمية تطلبها المنصة

مهم

يُفضّل التأكد من دقة البيانات لأن هذه المعلومات ستنعكس لاحقًا في الطلب النهائي والملخص.

بعد اكتمال بيانات ولي الأمر تبدأ مرحلة معلومات الطالب أو الطالبة.
الخطوة ٥

معلومات الابن / الابنة

تُعرض هنا بيانات الطالب أو الطالبة المطلوبة للتسجيل وربط الطلب بالمرحلة التعليمية المناسبة داخل المدرسة.

أمثلة على البيانات

  • الاسم والبيانات التعليمية
  • المعلومات المرتبطة بالمرحلة أو الصف
  • أي مرفقات أو بيانات إضافية تطلبها المدرسة

قبل المتابعة

راجع المدخلات بعناية قبل الانتقال إلى شاشة الملخص، لأن هذه الخطوة هي الأقرب للإرسال النهائي.

بعد هذه المرحلة يظهر الملخص النهائي للمراجعة.
الخطوة ٦

الملخص

هذه الشاشة تجمع البيانات النهائية لمراجعتها قبل الإرسال، وهي آخر مرحلة في رحلة التسجيل قبل تقديم الطلب رسميًا.

ما الذي يراجعه ولي الأمر؟

  • بيانات ولي الأمر
  • بيانات الابن / الابنة
  • صحة التسلسل قبل اعتماد الطلب

الخطوة النهائية

بعد التأكد من كل البيانات، يتم إرسال الطلب عبر المنصة ومتابعته حسب تحديثات المدرسة.
